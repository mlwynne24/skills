:sparkles: **Knowledge Hub — User Feedback Updates**

More updates to the Knowledge Hub functionality as of last week guys:

:card_file_box: **Curated Source Lists** — When chatting with the assistant, you can now save cited papers to a personal "My Sources" list and toggle a mode that restricts future queries to only those papers. Great for building focused evidence sets for synthesis.

:memo: **Restructured Annotations & Comments** — Annotations and comments are now separate sections. Annotations capture structured fields (outcome, data, methods, findings), while comments are a running thread for additional thoughts. Comments now appear as a visible history, whereas annotations are 1:1 with papers.

:brain: **Paper Summary Field** — A new "Paper Summary" annotation field lets you (or the AI) capture a short scene-setting summary of each paper. "Main Outcome" has also been renamed to "Main Outcome Variable" for clarity.

:broom: **Tidier Layout** — The Interact page has been moved to a more logical position in the navigation, and several small UI tweaks improve the overall flow when working with existing papers.
